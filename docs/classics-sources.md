# 古籍来源与收录策略

## 原则

- 古籍原作需属于公有领域，数字文本需有明确许可。
- 维基文库正文按 CC BY-SA 4.0 使用，Markdown 保留原页面链接和署名说明。
- “已收录”只代表来源已核验，不代表已经完成逐字校勘。
- 页面只有重定向、目录模板或极少正文时，不直接计入完整收录。
- 现代整理者增加的校注仍受数字文本许可约束，不改写为项目自有内容。
