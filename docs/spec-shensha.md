# 四柱神煞算法规格

## 目标

在现有紫微命盘旁增加一个“有出处的四柱神煞”面板，使用排盘已生成的四柱进行本地计算，并允许用户点击神煞名称检索对应古籍原文。

## 首批规则

| 神煞 | 取法 | 主要依据 |
|---|---|---|
| 天乙贵人 | 以日干查四柱地支 | 《三命通会》卷三·论天乙贵人 |
| 文昌贵 | 采用《三命通会》所载十干对应支位 | 《三命通会》卷三 |
| 禄神 | 以日干查禄位 | 《三命通会》卷三·论十干禄 |
| 阳刃 | 仅按甲、丙、戊、庚、壬五阳干查 | 《渊海子平》·论阳刃 |
| 驿马 | 年支、日支分别按三合局查四柱 | 《三命通会》卷三·论驿马 |
| 咸池 | 年支、日支分别按三合局查四柱 | 《三命通会》卷二·论咸池 |
| 将星 | 年支、日支分别取三合中位 | 《三命通会》卷二·论将星华盖 |
| 华盖 | 年支、日支分别取三合库位 | 《三命通会》卷二·论将星华盖 |
| 劫煞 | 年支、日支分别按五行绝处查四柱 | 《三命通会》卷三·论劫煞亡神 |
| 亡神 | 年支、日支分别按五行临官查四柱 | 《三命通会》卷三·论劫煞亡神 |
| 旬空 | 以日柱所在六旬推两支，并检查四柱 | 《三命通会》卷三·论空亡 |

## 产品边界

- 神煞没有公认的唯一现代标准，页面必须标明本项目采用的典籍和具体取法。
- 第一版只显示规则命中、对应柱位和来源，不输出确定性的吉凶、疾病、婚姻或人生结论。
- 紫微斗数星曜与四柱神煞分开展示，避免把两套体系混为一谈。
- 所有计算都在浏览器本地完成，不上传生辰。

## 交互

- 面板显示当前四柱、核验规则数量和命中的神煞。
- 每项显示命中的柱位与目标地支。
- 点击神煞名称，古籍搜索框切换到该词并打开相应原文结果。
- 页面保留一句简短提示：神煞只作辅助标记，不能脱离五行旺衰和整体格局单独断事。

## 验收

- 默认四柱 `庚午 辛巳 乙酉 甲申` 能命中天乙贵人、驿马、咸池、将星、亡神和旬空。
- 旬空计算覆盖六个旬首。
- 非法四柱输入返回清晰错误。
- 新面板可通过键盘访问，按钮有明确名称。
- 单元测试、组件测试、ESLint、生产构建和 Cloudflare dry-run 全部通过。
