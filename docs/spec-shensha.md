# 四柱神煞算法规格

## 目标

使用排盘已生成的年、月、日、时四柱和性别，在浏览器本地计算有明确古籍取法的神煞；每个命中结果必须显示柱位、查法、来源，并可打开对应古籍。

## 当前范围

版本 0.6.0 共核验 67 项规则，由首批 11 项扩充而来。

## 取法约定

1. 天乙、文昌、禄神、金舆、太极等干系规则，采用日干查四柱的页面口径。
2. 驿马、咸池、将星、华盖、劫煞、亡神、灾煞、六厄、攀鞍，分别以年支和日支查。
3. 天德、月德、德秀、月空、月厌、月煞、天赦、旌德、旌钺、四废，以月柱地支代表月令。
4. 元辰、勾神、绞神使用出生年干阴阳、出生年支和性别。
5. 旬空使用日柱所属六甲旬；截路空亡使用日干查时支。
6. 五鬼空亡、克害空亡、破祖空亡按古法年命口径，以年干查月、日、时。
7. 日贵、日德、魁罡、十恶大败等专日规则只检查日柱；金神只检查时柱。
8. 《三命通会》月德段数字文本末字疑有传抄误差，当前依同段“五行入垣”公式采用巳酉丑月见庚的通行表，并在代码测试中固定。

## 产品边界

- 神煞没有公认的唯一现代标准，不能宣传成“唯一权威算法”。
- 页面只显示规则命中和典籍依据，不自动输出疾病、婚姻、贫富、生死等确定性断语。
- 古籍所列“百二十神煞”中，部分规则需要胎元、命宫、纳音生克、大小运宫位，或原文自身认为穿凿；当前不做不完整推算。
- 紫微斗数星曜与四柱神煞分开展示。
- 所有计算都在访客浏览器本地完成，不上传生辰。

## 交互

- 面板显示核验规则数量、命中数量和日柱旬空。
- 命中项目按“贵人与文教、月令德曜、行旅与年神、日时专煞、刑冲杂煞”分组。
- 每项显示命中的柱位、具体查法和古籍章节。
- 点击名称打开站内对应古籍；点击出处链接打开维基文库原页。

## 验收

- 默认四柱 `庚午 辛巳 乙酉 甲申` 保留首批结果，并新增金舆、太极贵人、文星贵、红艳煞等命中。
- 春月 `戊寅` 日可命中天赦；春月 `庚申` 日可命中四废。
- `庚辰` 日、`癸酉` 时可命中魁罡、十恶大败和金神。
- 元辰、勾神、绞神能区分阳男、阴女与阴男、阳女。
- 旬空覆盖六个旬首，非法四柱返回中文错误。
- 所有命中都有分类、查法、来源 URL 和站内古籍 ID。
- 单元测试、组件测试、ESLint、生产构建和 Cloudflare dry-run 全部通过。
