# 古籍第三批扩充规格

## 目标

在不虚构缺失分卷、不收录重定向空页的前提下，补齐当前维基文库可稳定取得的《周易》《增删卜易》正文，并收录《卜筮正宗》现有版本的卷前材料。

## 已确认范围

- 《周易》六十四卦：64 篇。
- 《周易》传文：彖、大象、小象、文言、系辞上、系辞下、说卦、序卦、杂卦，共 9 篇。
- 《增删卜易》：序、1 至 26、15 又、26 又 1 至 4，共 32 篇。
- 《卜筮正宗（河潞武子龄校本）》：卷前 1 篇。
- 全站古籍文档由 29 篇增加到 134 篇。

## 数据边界

- 只导入维基文库 API 返回的实际正文。
- 拒绝重定向、正文过短和残留模板标记的页面。
- 《卜筮正宗》当前来源页虽列出卷一至卷十四，但这些分卷页面不可取得，因此只标注“卷前已收录”，不宣称全书完整。
- 所有新增文件保留原始页面地址、CC BY-SA 4.0 许可和“待逐字校勘”状态。

## 实现

- 在 `scripts/import-wikisource.mjs` 中维护明确页序。
- 为网络瞬断增加有限次数重试。
- 清理 MediaWiki 文件标记、分类、变体模板和多余标题格式。
- 已知极短的口诀页允许较低正文门槛，其余页面继续使用通用长度门槛。

## 验收

- 内容测试确认总数不少于 134 篇。
- 六十四卦必须恰好 64 篇。
- 所有文件必须含来源、许可和正文。
- 新增正文中不得残留重定向、文件模板或未闭合强调标记。
- `npm test`、`npm run lint`、`npm run build` 全部通过。
